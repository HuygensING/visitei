The VisiTEI project implements a hierarchical visitor approach to handling TEI.

Given an implementation VisitorImpl of Visitor the basic use case is as follows:

String xml = ...
Visitor visitor = new VisitorImpl();
Document document = Document.createFromXml(xml);
document.accept(visitor);

More sophisticated implementations may benefit from extending the DelegatingVisitor class, which allows handlers for specific elements to be registered. For example:

String xml = ...
DelegatingVisitor visitor = new MyDelegatingVisitor();
Document document = Document.createFromXml(xml);
document.accept(visitor);
String text = visitor.getContext().getResult();
See, for instance, RenderVisitor in the nl.knaw.huygens.tei.render package.
